# PR Practica

Este es un repositorio de practica para aprender el flujo de Pull Requests con Git y GitHub.

## Que es esto?

Aqui vas a aprender a:
- Crear branches
- Hacer commits
- Abrir Pull Requests

Es una herramienta muy util para colaborar en equipo, y te ayuda a recibir feedback antes de fusioner cambios al codigo principal.
